# Installing VS Code on macOS

## What is VS Code?

VS Code (Visual Studio Code) is a free program that helps us write and run code.

---

## Step 1: Download VS Code

1. Open Safari or Chrome.
2. Go to:

   https://code.visualstudio.com

3. Click **Download for macOS**.

Wait for the download to finish.

---

## Step 2: Install VS Code

1. Open the **Downloads** folder.
2. Double-click the downloaded VS Code ZIP file.
3. Drag **Visual Studio Code** into the **Applications** folder.

That's it! VS Code is now installed.

---

## Step 3: Open VS Code

1. Open the **Applications** folder.
2. Double-click **Visual Studio Code**.

You should see a window like this:

- A menu at the top
- A large editing area in the middle
- A toolbar on the left side

🎉 Congratulations! VS Code is ready to use.

---

## Step 4: Create Your First File

1. Click **File → New Text File**.
2. Type:

```java
System.out.println("Hello World!");
```

3. Click **File → Save**.
4. Save the file as:

```
Hello.java
```

---

## Step 5: Install Java Support

To write Java programs, we need a Java extension.

1. Click the **Extensions** icon on the left side.
2. Search for:

```
Extension Pack for Java
```

3. Click **Install**.

Wait for the installation to finish.

---

## Checkpoint

By the end of this lesson, you should be able to:

- ✅ Install VS Code
- ✅ Open VS Code
- ✅ Create and save a file
- ✅ Install the Java extension

You are now ready to start learning Java programming!
