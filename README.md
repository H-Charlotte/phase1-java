# Phase 1 - Java Fundamentals & Development

A comprehensive Java tutoring program designed for middle school and high school students participating in robotics competitions (FTC, FRC, VEX).

## Overview

This course focuses on building **software engineering thinking** and practical robotics programming skills through hands-on learning. Students learn Java fundamentals while developing debugging abilities and AI-assisted development workflows.

## Course Goals

By completing Phase 1, students will:

- Understand computers, software, and modern development workflows
- Learn beginner Java programming fundamentals
- Build strong debugging and problem-solving skills
- Learn to work effectively with AI development tools
- Develop engineering-oriented programming habits
- Prepare for robotics software development

## Who This Is For

- Students in middle school or high school
- Participants in FTC, FRC, or VEX robotics competitions
- Anyone interested in robotics, AI, automation, and software engineering
- **No prior programming experience required**

## Getting Started

### 1. Setup Your Environment

Follow the preparation guides in order:

1. [Git Setup](preparation/git.md) - Learn version control
2. [Homebrew Setup](preparation/homebrew.md) - Install package manager (macOS)
3. [Java Installation](preparation/java.md) - Install Java JDK
4. [VS Code Setup](preparation/vscode.md) - Configure your IDE
5. [General Guide](preparation/guide.md) - Additional setup help

### 2. Start with Lesson 1

Begin with the fundamentals:
- Development environment setup
- Introduction to Computer Science
- Programming basics and your first program

## How to Use This Repository

1. **For Students**: Follow the lessons in order. Start with preparation, then work through each lesson's projects.

2. **For Instructors**: Use the lesson files and projects as teaching materials. Each project builds on previous concepts.

3. **For Self-Learners**: Progress through the curriculum at your own pace, ensuring you understand each concept before moving forward.

## Tips for Success

1. **Type, don't copy-paste** - Writing code by hand helps learning
2. **Break and fix code intentionally** - Learn from mistakes
3. **Ask questions early** - Don't get stuck alone
4. **Debug regularly** - Use print statements and debuggers
5. **Practice outside lessons** - Write your own small programs
6. **Use AI tools wisely** - They're helpers, not replacements for understanding
