# Phase 2: FTC / FRC Robotics Development

## Goals

Students will learn:

- FTC / FRC software architecture
- Robot control systems
- Sensors and actuators
- Competition debugging workflows
- Team collaboration

---

## Lesson 1: FTC / FRC Software Structure

### Topics

- FTC / FRC project structure
- OpMode concepts
- TeleOp vs Autonomous
- RobotContainer / Subsystem concepts
- Packages and classes

### Focus

Why large projects cannot live inside one file.

---

## Lesson 2: Robotics Debugging

### Common Problems

- Incorrect motor directions
- Sensor reading issues
- Logic bugs
- Autonomous failures

### Debugging Topics

- Breakpoints
- Watch windows
- Step into
- Call stack

### Goal

Students learn how to locate and isolate problems systematically.
