# Computer Science, Robotics, Java, and AI Training Program

## Course Objectives

This program is designed for middle school and high school students participating in robotics competitions such as FTC, FRC, and VEX.

The goal is not test preparation. The goal is to help students build:

- Software engineering thinking
- Robotics programming skills
- Debugging ability
- AI-assisted development skills
- Team collaboration experience
- Long-term self-learning ability

### Course Characteristics

- Faster pace than traditional beginner programming classes
- Emphasis on reading, modifying, and debugging existing code
- Strong focus on engineering thinking instead of memorizing syntax
- Heavy emphasis on debugging and problem-solving
- Early introduction to AI development tools
- Closely aligned with real robotics competition workflows

### Core Skills

- Understanding how robot software works
- Reading and understanding existing code
- Building engineering-oriented programming habits
- Learning Java and Python development
- Debugging robot software
- Understanding sensors, input/output, and control systems
- Learning software collaboration workflows
- Using AI tools for development assistance
- Participating in robotics software projects

---

# Student Profile

Suitable for students who:

- Participate in FTC, FRC, or VEX robotics competitions
- Are interested in robotics, AI, automation, and software engineering
- Want to improve debugging and programming skills
- Are currently in middle school or high school

No prior experience required:

- No programming background required
- No competitive math background required
- No algorithm experience required

Recommended:

- Separate beginner and advanced groups if possible

# Overall Course Structure

| Phase   | Topic                          | Core Goal                                                                              | Details         |
| ------- | ------------------------------ | -------------------------------------------------------------------------------------- | --------------- |
| Phase 1 | Fundamentals + Java            | Build computer science and software development foundations                            | [phase1.md](phase1.md) |
| Phase 2 | FTC / FRC Robotics Development | Learn robotics programming and engineering workflows                                   | [phase2.md](phase2.md) |
| Phase 3 | Python Fundamentals            | Learn Python fundamentals and understand how Python connects to modern AI technologies | [phase3.md](phase3.md) |

### Schedule

- 5 classes per phase
- Each class: 2 hours
- Each lesson is divided into Part 1 and Part 2
- Structure:
  - Part 1: 50 minutes instruction
  - 10 minutes break
  - Part 2: 50 minutes instruction
  - 10 minutes Q&A and discussion
- Every lesson includes:
  - Pre-class Preparation
  - In-class Practice
  - Post-class Assignment
- Total: 15 classes / 30 hours

# Topics Not Recommended for Beginners

Avoid going too deep into:

- Generics
- Stream API
- Maven
- Spring
- JVM internals
- Advanced design patterns
- Competitive programming theory

The goal is to build:

- Interest
- Engineering confidence
- Practical development ability
