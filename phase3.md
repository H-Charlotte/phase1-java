# Phase 3: Python + AI Applications

## Goals

Students will learn how Python is used in:

- AI systems
- Automation
- Robotics workflows
- Data analysis
- AI-assisted engineering

---

## Lesson 1: Python Fundamentals

### Topics

- Python environment setup
- Basic syntax
- Input and output
- Lists and dictionaries
- Functions

---

## Lesson 2: Python Automation

### Topics

- File processing
- Batch processing
- Automation scripts
- Workflow automation concepts

### Exercises

- Batch file renaming
- Data organization

---

## Lesson 3: AI APIs and Tools

### Topics

- ChatGPT API concepts
- AI agents
- Prompt engineering
- AI workflows

### Focus

Students understand that AI is not only a chatbot, but also a development tool.

---

## Lesson 4: Python for Robotics

### Topics

- Data logging
- Computer vision concepts
- Robot data analysis
- Dashboard concepts

---

## Lesson 5: AI Mini Projects

### Suggested Projects

- AI data analysis tools
- Automation tools
- Simple AI agents
- Robot log analysis tools

### Focus

- AI-assisted development
- Debugging
- Engineering organization
